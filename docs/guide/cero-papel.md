# Cero Papel

IesApp contribuye en múltiples direcciones a la eliminación del papel en la labor del centro.

Esto se logra evitando el uso de formatos impresos y guardando los documentos digitalizados:

- Se envían circulares y notificaciones por email.
- Las facturas y albaranes se guardan digitalizados.
- La justificación de ausencias se gestiona de forma digital.
- Incluso se da soporte para digitalizar el expediente del alumno.
