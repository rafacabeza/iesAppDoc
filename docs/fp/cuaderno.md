# Cuaderno del profesor

El profesorado necesita apoyarse en un cuaderno de aula para completar su labor docente. Las necesidades pueden ser variables para distintos profesores e incluso para un mismo profesor en distintos grupos. No se ha considerado la posibilidad de llevar todas esas informaciones, sino que se han incorporado las cuestiones que marca el Sistema de Gestión de la Calidad y que requerían el uso de un formalismo concreto, unos formatos, que a día de hoy están superados:

**Programación**

El cuaderno parte de varias informaciones para programar todo el curso:
- Horario del profesor para esa materia.
- Calendario de inicio, fin y de fiestas.
- Relación de unidades didácticas y la duración de las mismas. Esto se hace de acuerdo a las semanas establecidas en el currículum.
- También se tiene en cuenta si existe subdivisión del grupo para dar clases con medio grupo.

Una vez dada esa información se programa el curso de manera que podemos saber qué unidad trabajar en cada una de las sesiones del curso, incluso si tenemos días de holgura o si por el contrario debemos condensar algunas unidades por las necesidades del calendario.

La programación se representa como un diagrama de Gantt y a su vez se pueden identificar cada una de las sesiones programadas durante el curso.

**Seguimiento**

Una vez realizada la programación es el momento de reflejar el seguimiento de las clases. En el cuaderno vemos cada sesión programada y la unidad didáctica que deberíamos impartir. Reflejar el seguimiento supone:

- Indicar si se ha impartido esa unidad u otra, o incluso si la sesión se ha cancelado (actividades complementarias, huelgas, ausencia del profesor, ...)
- De manera opcional podemos hacer anotaciones sobre lo que ha ocurrido en la clase.
- El seguimiento registrado se incorpora en el diagrama de Gantt, resumen de la actividad docente.
- Cada unidad tiene un peso específico en relación a las horas asignadas por lo que podemos saber qué porcentaje de cada unidad se ha impartido en cada momento.

Realizado este seguimiento, el cuaderno registra hitos mensuales  de seguiento que recogen:

- Porcentaje de horas impartidas en relación a las programadas.
- Porcentaje de contenidos global que se ha impartido al final de cada mes.

**Calificaciones**

Las calificaciones no se registran en el cuaderno. No se consideró viable por las diferentes estrategias para llevarlo a cabo. Lo que sí se recoge es el resumen de las mismas para saber el porcentaje de aprobados tras cada evaluación. Este es un indicador de Calidad para el proceso de la labor docente.

**Memoria**

Al llegar junio hay que hacer una valoración del curso. La memoria permite volorar de forma numérica y descriptiva varios aspectos de la labor realizada y de las dificultades encontradas.
