# Introducción
**IesApp y la FP**

Aunque IesApp no nació para dar soporte específico a las cuestiones de FP y del sistema de Gestión de la Calidad, se han ido incorporando funcionalidades que son muy útiles en ambos terrenos:

- Encuestas.
- Cuaderno del profeor. Que sirve para:
    - Programación de los módulos.
    - Seguimiento de las programaciones.
    - Memoria de los módulos
- Formación en empresas:
    - Gestión histórica de las empresas. Información histórica.
    - Listado de prácticas de cada curso como base para la encuestas a las empresas.
- Gestión de las pérdidas de evaluación continua
- Gestión de las  bajas por inasistencia
