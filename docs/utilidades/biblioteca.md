# Biblioteca

Para terminar, IesApp permite gestionar la biblioteca del IES Santiago Hernández.

Por cuestiones históricas, la biblioteca se gestionaba mediante una pequeña aplicación de access.

Llegó el momento de apagar ese sistema que había quedado obsoleto y se optó por incluir la biblioteca como otro módulo más dentro de IesApp. Los posibles lectores son los usuarios de IesApp y la integración parecía una buena decisión. Tal vez sea la funcionalidad menos exportable a otros centros pero en nuestro centro de origen cumple una importante labor.
