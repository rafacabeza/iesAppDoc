# Libros de texto

La elaboración de listados de libros de texto es una tarea que implica al equipo directivo y a todos los jefes de departamento. Una tarea realizada entre muchos y que requiere una coordinación que no siempre se logra.

IesApp permite elaborar estas listas de libros en las siguientes condiciones: 
- Los jefes de departamento son los que actualizan los libros de sus materias. Se pueden exportar de un curso para el siguiente.
- Cada libro lleva la etiqueta del primer curso en el que se utilizó para llevar una renovación de acuerdo a la normativa.
- Los alumnos ven los libros de acuerdo a su matrícula.
- Se generan las listas de libros para elaborar lotes del banco de libros o para que los alumnos puedan comprarlos en la librería. 
- El listado de libros es accesible como invitado y está enlazado a la web del centro para que las librerías tengan siempre la versión actualizada.
