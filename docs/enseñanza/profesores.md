# Profesores

IesApp permite mantener la información completa del claustro. Es un sistema de directorio que permite a todo el claustro localizar el correo corporativo de los compañeros, identificarlo mediante la orla de fotos.

**Plantilla**

Además de ese servicio sencillo de directorio nos permite definir la **plantilla** actual en cada momento. Esta información sólo es visibe para jefatura. La plantilla está definida para cada curso y cada profesor ocupa una plaza de forma fija, de forma interina (de plantilla sin titular, de cupo sin titular o reserva del titular) o mediante sustitución. Esta información que esbásica no está en ningún sitio de los sistemas oficiales.

**Horario**

La información fundamental que tenemos de cada profesor es su horario. Los horarios se importan con el mismo fichero que genera Peñalara para SIGAD. Es una pieza fundamental para el funcionamiento del centro:
- Están siempre actualizados y disponibles para todo el claustro y conserjería.
- Son la base del sistema de guardias
- IesApp también ayuda con la modificación de horarios (FCT/FEOE) de forma coordinada entre jefatura y los profesores interesados.
