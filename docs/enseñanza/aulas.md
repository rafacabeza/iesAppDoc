# Aulas

La información de aulas es bastante reducida. No va más allá de un nombre y descripción pero es la base para la gestión de horarios y reserva de espacios:

**Horarios**

Los horarios de las aulas está disponible para todo el personal del centro. Incluidos conserjes y limpiadores.

Más allá del uso por parte del profesorado, es fundamental cuando conserjería debe localizar a un alumno o para que las limpiadoras sepan la ocupación del aula desde el día que se inician las clases.

**Reservas y espacios libres**

Apoyado en el sistema de horarios, el profesorado puede realizar reserva sobre las aulas libres, algo fundamental en algunos espacios de todos los centros.

Con la información de horarios y reservas tenemos las bases para localizar qué aulas hay libres en un momento determinado. Y esto está disponible para todo el profesorado, no sólo para jefatura.
