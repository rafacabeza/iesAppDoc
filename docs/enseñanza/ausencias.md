# Ausencias

La gestión de ausencias es fundamental para el sistema de guardias. Durante mucho tiempo ése era su única finalidad pero actualmente también se usa para ayudar a jefatura en la gestión de la justificación de las mismas.

SIGAD gestiona las ausencias del profesorado pero sigue una filosofía anticuada en la que el jefe de estudios es un burócrata que procesa esta información para hacerla disponible para la Inspección Educativa.

Si sólo usamos SIGAD, gestionar las ausencias es un esfuerzo ímprobo y una cantidad de documentación bastante abultada en un centro mediano o grande.

Vamos a repasar el proceso:

- En IesApp las ausencias se registran para el sistema de guardias. Lo hace Jefatura de Estudios. Es una decisión política que no lo pueda hacer el propio profesor.
- El profesorado ve una lista de sus ausencias de todo el curso. La justificación de las mismas se realiza rellenando el motivo de la ausencia y aportando un justificante escaneado.
- Una vez justificada la ausencia, jefatura de estudios revisa la justificación, la valida y la pasa a SIGAD.
