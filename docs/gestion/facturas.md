# Gestión de facturas y documentos contables

IesApp aporta un sistema de gestión de documentos contables, base para realizar los pagos y para la contabilidad. Los objetivos son:
- Que todos las facturas y demás documentos contables estén digitalizados y sean fácilmente localizables. A cada factura se le puede adjuntar uno o varios PDF.
- Que permitan marcar qué facturas pueden pagarse en cada momento.
- Que sirva de apoyo a la labor contable de manera que podemos identificar los documentos como pendientes de contabilizar o contabilizados. Además, apoyado en la tesorería, el pago o cobro puede estar en distintos estados (borrador, visto bueno, pagado, en espera, contabilizado). 

Las facturas asociadas a una compra están enlazadas de manera que de una se puede ir a la otra y viceversa.
