# Gestión de archivo: Entradas y Salidas

Los centros no disponen de un sistema para gestionar el archivo de entradas y salidas. Durante muchos años se ha usado IES2000 en multitud de centros. Dado que esa aplicación está prácticamente olvidada, se incorporó esta función en IesApp.

Se registran y numeran por separado las entradas y salidas, con numeración correlativa que se reinicia cada año.

Se puede digitalizar el documento en cuestión y subirlo a la plataforma.
